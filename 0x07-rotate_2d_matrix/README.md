# 0. Rotate 2D Matrix

## Description
This project requires you to implement an in-place algorithm to rotate an n x n 2D matrix by 90 degrees clockwise. The challenge emphasizes matrix manipulation and optimizing space complexity by performing operations directly on the original matrix.

## Concepts
- **Matrix Representation**: Understanding 2D matrices using lists of lists in Python.
- **In-place Operations**: Modifying the matrix without creating additional data structures.
- **Matrix Transposition**: Swapping rows with columns as a key step in the rotation.
- **Reversing Rows**: Reversing each row after transposition to complete the rotation.

## Steps
1. **Transpose the Matrix**: Convert rows to columns.
2. **Reverse Each Row**: Reverse the elements in each row.
